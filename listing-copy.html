App name

Timesheet Basic

Tagline

Track personal and team Jira worklogs with fast timesheet views.

Summary

Timesheet Basic helps Jira teams log, review, and analyze worklogs through personal and team timesheets plus issue-level logging.

Description

Timesheet Basic is a Forge-powered Jira app that makes worklog tracking easier for individuals and teams.

  • My Timesheet: a personal view of logged time with quick summaries and CSV export.
  • Team Timesheet: a compact person-by-day grid to review team activity.
  • Issue Panel Logging: add worklogs directly from Jira issue view.

Features

  • Personal timesheet dashboard
  • Team timesheet table (expandable by person/day)
  • Jira issue panel time logging
  • CSV export for personal worklogs
  • Native Jira worklog API integration

Permissions justification

  • read:jira-user — required to resolve user identities for timesheet displays.
  • read:jira-work — required to read issue and worklog data.
  • write:jira-work — required to create, update, and delete worklogs from the app.
